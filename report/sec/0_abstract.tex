\begin{abstract}
  
While recently, Vision Transformers (ViTs) achieve state-of-the-art performance, their quadratic self-attention mechanism causes immense computational costs. Recent linear attention mechanisms mitigate the quadratic bottleneck but still redundantly process uninformative background tokens, wasting computational budget and limiting the model accuracy. In this paper, we propose a Multi-Agent Reinforcement Learning (MARL) gating framework to assign the importance of each token. Each visual token acts as an autonomous agent, utilizing a shared Actor-Critic policy to output continuous survival weights, gating information flow into the recurrent memory.
