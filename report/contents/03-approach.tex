\subsection{Proposed Method}
Standard ViTs rely on softmax attention which causes quadratic complexity.  To tackle this,
we adopt linear attention as a replacement to reduce to linear complexity. Specifically, we adopt
the RALA architecture from Qihan Fan \cite{fan2025breaking}. We extend the method by combining a token
pruning mechanism, which aim to refine the input for linear attention, improving its performance.
In detail, at intermediate layer depth in the ViT, we use a Multi-Agent Proximal Policy Optimization, as proposed
in \cite{lu2025reinforcement} where each visual token is considered a specific agent, to drop low value token. The surviving token is then
fed into the linear attention layer.
