\section{Method}
We augment a linear-attention ViT with a continuous Actor-Critic routing module that operates jointly over two recurrent axes: spatial chunks within each layer, and depth across layers. The routing policy observes both what the current layer has accumulated so far and what the previous layer retained globally, enabling depth-aware token suppression without any explicit layer-index embedding.

%------------------------------------------------------------------
\subsection{Architecture Overview}
%------------------------------------------------------------------
An input image is tokenised into $N$ patch embeddings and partitioned into $T$ sequential chunks of size $C$. The backbone has $L$ transformer layers; each layer $\ell \in \{1,\dots,L\}$ maintains its own recurrent memory $(S_{\ell,t},\, Z_{\ell,t})$ initialised to zero at the start of that layer. At each step $(\ell, t)$, the Actor-Critic observes a causal state encoding local features, the within-layer memory accumulated so far, the \emph{completed} memory from the previous layer, and the remaining compute budget. It emits continuous per-token weights $w_{\ell,t}$ that gate the memory update, softly suppressing uninformative tokens. After all $T$ chunks, the completed memory $(S_{\ell,T}, Z_{\ell,T})$ is passed to layer $\ell{+}1$ as cross-layer context. The final memory $(S_{L,T}, Z_{L,T})$ feeds a task head whose output drives the terminal RL reward.

Training follows a two-phase curriculum. \textbf{Phase~1} bootstraps the Actor and Critic from Fisher Information saliency targets, avoiding representation collapse when RL is applied to a randomly initialised policy. \textbf{Phase~2} fine-tunes the full system end-to-end via PPO with Generalized Advantage Estimation. At inference, stochastic sampling is disabled; weights below a hard threshold $\tau$ physically purge tokens from the GPU buffer, yielding exact FLOP reductions.

\subsection{Chunkwise Linear Attention}
%------------------------------------------------------------------
Standard linear attention exploits the associativity of the kernel inner product to achieve $\mathcal{O}(N)$ complexity. For the $i$-th query:
\begin{equation}
    O_i = \frac{\phi(Q_i)\!\sum_{j=1}^{N}\phi(K_j)^\top V_j}
               {\phi(Q_i)\!\sum_{j=1}^{N}\phi(K_j)^\top}
\end{equation}

We reformulate this as a causal chunkwise recurrence. Within layer $\ell$, the sequence is partitioned into $T$ chunks of size $C$. Two memory accumulators are maintained:
\begin{equation}
    S_{\ell,t} = S_{\ell,t-1} + \phi(K_{\ell,t})^\top V_{\ell,t}, \qquad
    Z_{\ell,t} = Z_{\ell,t-1} + \sum_{i \in t}\phi(K_{\ell,i,t})^\top
\end{equation}
with $S_{\ell,0} = \mathbf{0}$ and $Z_{\ell,0} = \mathbf{0}$ at the start of each layer. The attention output for chunk $t$ at layer $\ell$ is:
\begin{equation}
    O_{\ell,t} = \frac{\phi(Q_{\ell,t})\,S_{\ell,t}}{\phi(Q_{\ell,t})\,Z_{\ell,t}}
\end{equation}

This double-indexed recurrence creates a two-dimensional trajectory $(\ell, t)$ that the Actor-Critic acts upon, enabling routing decisions that are sensitive to both spatial position within a layer and depth in the network.

%------------------------------------------------------------------
\subsection{Markov Decision Process Formulation}
%------------------------------------------------------------------
Each step $(\ell, t)$ is a node in a discrete-time MDP. The agent processes all $T$ chunks of layer $\ell$ before advancing to layer $\ell{+}1$, at which point the completed memory $S_{\ell,T}$ becomes available as cross-layer context.

\subsubsection{State Space ($\mathcal{S}$)}
At step $(\ell, t)$, the agent observes:
\begin{equation}
s_{\ell,t} = \Bigl[
  K_{\ell,t}
  \;\|\;
  \mathrm{Proj}(S_{\ell,t-1})
  \;\|\;
  \mathrm{Proj}(Z_{\ell,t-1})
  \;\|\;
  \mathrm{Proj}(S_{\ell-1,T})
  \;\|\;
  B_{\ell,t}
\Bigr]
\end{equation}

\begin{itemize}
  \item \textbf{Local features} $K_{\ell,t} \in \mathbb{R}^{C \times D}$: visual features of the current chunk at the current layer.
  \item \textbf{Within-layer memory} $\mathrm{Proj}(S_{\ell,t-1}) \in \mathbb{R}^{D}$: a learned linear projection of the memory accumulated so far in layer $\ell$, encoding what this layer has already seen.
  \item \textbf{Normalization projection} $\mathrm{Proj}(Z_{\ell,t-1}) \in \mathbb{R}^{D}$: projection of the normalization accumulator, encoding the effective token mass retained so far.
  \item \textbf{Cross-layer context} $\mathrm{Proj}(S_{\ell-1,T}) \in \mathbb{R}^{D}$: a learned projection of the \emph{completed} memory from the previous layer, providing a summary of which tokens the network considered globally important at depth $\ell{-}1$. For $\ell=1$ this term is zero.
  \item \textbf{Dynamic budget} $B_{\ell,t} \in [0,1]$: remaining compute budget, decremented by the weights emitted at each step.
\end{itemize}

The inclusion of $\mathrm{Proj}(S_{\ell-1,T})$ is the key addition over single-axis routing: the agent at layer $\ell$ can reinforce attention to tokens that were salient in layer $\ell{-}1$, or deliberately shift focus to complementary structure, without any explicit depth index.

\subsubsection{Action Space ($\mathcal{A}$)}
For each chunk of size $C$, the Actor outputs independent per-token weights:
\begin{equation}
a_{\ell,t} = \{w_{1,\ell,t},\, w_{2,\ell,t},\, \dots,\, w_{C,\ell,t}\},
\quad w_{i,\ell,t} \in (0,1)
\end{equation}
\begin{equation}
a_{\ell,t} \sim \mathrm{SquashedGaussian}\!\left(\mu_\theta(s_{\ell,t}),\,\sigma_\theta(s_{\ell,t})\right)
\end{equation}
Weights are independent and do not sum to one, allowing the agent to suppress an entire chunk by assigning near-zero weights to all its tokens.

\subsubsection{Transition Dynamics}
The within-layer memory update gates token contributions by their weights:
\begin{equation}
S_{\ell,t} = S_{\ell,t-1}
  + \left(w_{\ell,t} \odot \phi(K_{\ell,t})\right)^\top V_{\ell,t}
  + \gamma \cdot (1-\bar{w}_{\ell,t}) \cdot \frac{S_{\ell,t-1}}{t-1}
\end{equation}
where the State Dilution term (zero for $t=1$) replenishes the memory with its historical mean scaled by the suppression degree $(1-\bar{w}_{\ell,t})$, preventing stagnation when the agent prunes aggressively. The normalization accumulator evolves as:
\begin{equation}
Z_{\ell,t} = Z_{\ell,t-1} + \sum_{i=1}^{C} w_{i,\ell,t}\,\phi(K_{\ell,i,t})^\top
\end{equation}
where $\phi(\cdot)$ is the linear attention feature map and $\odot$ denotes element-wise scaling. The budget is decremented as:
\begin{equation}
B_{\ell,t+1} = B_{\ell,t} - \frac{1}{N}\sum_{i=1}^{C} w_{i,\ell,t}
\end{equation}
At the end of layer $\ell$ (i.e.\ $t=T$), the completed state $S_{\ell,T}$ is passed as cross-layer context to the next layer, and the within-layer memory is reset: $S_{\ell+1,0} = \mathbf{0}$.

\subsubsection{Reward Function ($\mathcal{R}$)}
A single terminal reward is provided after all $L$ layers and $T$ chunks have been processed:
\begin{equation}
R = \mathrm{TaskMetric} - \lambda_{\text{penalty}} \cdot \max(0,\, 1 - B_{L,T})
\end{equation}
where \textbf{TaskMetric} is the downstream performance measure (e.g.\ mAP or top-1 accuracy) and the penalty term discourages exceeding the allocated token budget.

%------------------------------------------------------------------
\subsection{Squashed Gaussian Actor-Critic}
%------------------------------------------------------------------
We model the routing policy $\pi_\theta(a_{\ell,t} \mid s_{\ell,t})$ as a Squashed Gaussian, avoiding the instability of discrete gates and the competitive coupling of a global Softmax. The Actor outputs the parameters of an unbounded Normal distribution:
\begin{equation}
\mu_\theta = \mathrm{MLP}_\mu(s_{\ell,t}), \qquad
\sigma_\theta = \mathrm{Softplus}\!\left(\mathrm{MLP}_\sigma(s_{\ell,t})\right) + \epsilon
\end{equation}
Actions are sampled via the reparameterization trick and squashed to $(0,1)$:
\begin{equation}
w_{\ell,t} = \frac{\tanh(x)+1}{2}, \qquad x \sim \mathcal{N}(\mu_\theta, \sigma_\theta^2)
\end{equation}
The Tanh transformation changes the probability density; the log-probability is corrected by the Jacobian of the transformation:
\begin{equation}
\log\pi(w_{\ell,t}) = \log p(x) - \sum_i \log\!\left(1 - \tanh^2(x_i) + \epsilon\right)
\end{equation}
A Value network (Critic) $V_\phi(s_{\ell,t})$ estimates the expected episodic return from state $s_{\ell,t}$, providing a baseline for PPO updates.

%------------------------------------------------------------------
\subsection{Phase 1: Scale-Aligned Fisher Distillation}
%------------------------------------------------------------------
Applying RL to a randomly initialised policy in a high-dimensional visual state space leads to representation collapse: random weights destroy the recurrent memory before any useful gradient signal propagates. Phase~1 circumvents this by supervising the Actor and Critic from an empirical Fisher Information oracle.

After warming up the backbone on a dense classification task, we compute the squared gradient of the cross-entropy loss with respect to each token's features as a proxy for its informational utility $\mathcal{I}_{i}$. Spatial smoothing is applied to preserve object boundaries, and scores are min-max normalised to a strict $\tilde{\mathcal{I}}_{i} \in [0,1]$ range to prevent value-scale mismatch during the subsequent RL phase.

The Actor and Critic are then trained by supervised regression:
\begin{equation}
\mathcal{L}_{\text{Actor}} = \mathbb{E}\!\left[\left(w_\mu(s_{\ell,t}) - \tilde{\mathcal{I}}_{\ell,t}\right)^2\right]
\end{equation}
\begin{equation}
\mathcal{L}_{\text{Critic}} = \mathbb{E}\!\left[\left(V_\phi(s_{\ell,t}) - \sum_{i \in t}\tilde{\mathcal{I}}_{\ell,i}\right)^2\right]
\end{equation}
This initialises the Actor to approximate gradient-based saliency and the Critic to predict chunk-level utility, providing a stable warm start for Phase~2.

%------------------------------------------------------------------
\subsection{Phase 2: Task-Specific PPO Fine-tuning}
%------------------------------------------------------------------
In Phase~2, the full system is transferred to the downstream task and optimised end-to-end. The two-dimensional MDP trajectory $\{(\ell, t)\}_{\ell=1,t=1}^{L,T}$ is rolled out per image; GAE computes advantages $\hat{A}_{\ell,t}$ from the Critic's value estimates. The Actor is updated via the clipped PPO surrogate with an entropy bonus over the true squashed distribution:
\begin{equation}
\begin{split}
\mathcal{L}_{\mathrm{PPO}} =\; &{-}\mathbb{E}\!\left[
  \min\!\left(r_t(\theta)\hat{A}_{\ell,t},\;
  \mathrm{clip}(r_t(\theta),1\pm\epsilon)\hat{A}_{\ell,t}\right)\right] \\
&- \lambda_{\mathrm{ent}}\,H(\pi_\theta)
\end{split}
\end{equation}
where $H(\pi_\theta)$ is the entropy of the squashed Gaussian policy:
\begin{equation}
H(\pi_\theta) = \tfrac{1}{2}\log(2\pi e\,\sigma_\theta^2)
+ \mathbb{E}\!\left[\sum_i \log\!\left(1 - \tanh^2(x_i) + \epsilon\right)\right]
\end{equation}
Using the true squashed entropy rather than the pre-squash Gaussian entropy prevents the degenerate incentive of pushing weights toward the saturation boundaries of the Tanh nonlinearity.

During inference, stochastic sampling is disabled. The Actor outputs deterministic weights; tokens with $w_{\ell,t} \leq \tau$ are physically purged from the GPU memory buffer via a \texttt{nonzero()} packing operation, yielding exact FLOP reductions. Surviving tokens retain their continuous scaling factors in the linear attention update, bypassing the quantisation shock of discrete binary gates.
