\section{Introduction}
In recent years, Vision Transformer is one of the leading architectures in Computer Vision.
However their main drawback is high computational cost \(O(n^{{2}})\) causing by the softmax attention. 
To address this issues, many works attempted to replace the self attention core with linear attention mechanisms, which help reduce the quadratic complexity.
On the other hand, there's a different perspective from the Reinforcement Learning field, in which a agent is used to efficiently select unnecessary visual token inside a Vision Transformer, helping reduce the computational cost. In this paper, we proposed a combined method of the two, and then carry out tests on a object detection task.
