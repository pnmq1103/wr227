\subsection{Linear Attention} 
%Breaking the Low-Rank Dilemma of Linear Attention  
Linear Attention[cite original linear attention] fixes the quadratic complexity of Softmax attention by using kernel functions to approximate the \(Sim()\) operator, reducing the complexity to linear. However, its performance is still largely behind Softmax. Many recent works try to tackle this issue[cite all linear attention].
FLattenTransformer [23] introduces a focused function to enhance the ability of linear attention to concentrate on relevant information, while MLLA [24] establishes a link between linear attention and the Mamba model [19]. 
 More recently, Qihang Fan et al.[break low rank dilemma] addresses a fundamental limitation of linear attention by identifying the low-rank nature of its feature representations as a major cause of performance degradation. To overcome this, the paper proposes Rank-Augmented Linear Attention, which enriches both the key–value buffer and output features to raise their effective rank. Many other approaches either rely on kernel-based approximations of the Softmax function or integrate depthwise convolution to improve feature diversity.
